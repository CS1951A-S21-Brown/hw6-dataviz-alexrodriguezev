(10 points) My dashboard answers the questions by providing the information in a manner that is easily and quickly understood. To answer how many titles per genre on are Netflix, I provided a bar graph that provided each genre and its corresponding numnber of titles. Since the plot has a wide range, I provided buttons to filter by type, so the user could view just the movie titles and just the TV show titles. To address the question about average runtime of movies by release year, I created a scatterplot that plots release years and the average runtime of all the movies released that year. A user can hover over datapoints to see more detailed information, if the graph axes are too hard to read. Finally, to address the question about learning more about movie casts, I created a flow chart that connects all the actors that have played in movies together. To make the graph more relevant and easy to read, I included only the top actors of each year as a measure of how much movies they were in in total.
